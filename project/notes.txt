
Initialise two Debian Unstable VM images

$ debvm-create -h client -k keys.pub -o client.ext4 -r unstable -z 1G -- --include curl,locales
$ debvm-create -h server -k keys.pub -o server.ext4 -r unstable -z 1G -- --include curl,locales

Snapshot images
$ cp client.ext4 client-snapshot.ext4
$ cp server.ext4 server-snapshot.ext4

Start VMs

$ debvm-run -i client.ext4 -s 2222
$ debvm-run -i server.ext4 -s 2223

Connect over SSH

$ ssh -o NoHostAuthenticationForLocalhost=yes root@127.0.0.1:2222
$ ssh -o NoHostAuthenticationForLocalhost=yes root@127.0.0.1:2223
